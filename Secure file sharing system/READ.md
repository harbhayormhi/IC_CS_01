# 🔐 Secure File Sharing System

A secure web-based file sharing system built with Python and Flask that enables users to upload, encrypt, and share files safely.

This project demonstrates how cryptography and secure authentication mechanisms can be implemented in a real-world application to protect sensitive data during file transfer.

---

## 📌 Project Overview

The Secure File Sharing System is designed to protect files from unauthorized access using modern cryptographic techniques.

The system ensures that:
- Files are encrypted before storage
- Encryption keys are securely exchanged
- Users must authenticate before accessing files

This project demonstrates key cybersecurity concepts such as encryption, authentication, and secure communication.

---

## 🚀 Features

- 🔑 User Authentication with hashed passwords
- 📁 Secure File Upload and Download
- 🔒 AES Encryption for protecting files
- 🔐 RSA Encryption for secure key exchange
- 🛡️ Access control for authorized users
- 🌐 Secure communication using HTTPS / encrypted sockets
